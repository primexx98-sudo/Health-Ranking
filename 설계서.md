## 6. 크롤러 방식 (플랫폼별 핵심)

| 플랫폼 | 라이브러리 | 추출 방식 | Cloudflare |
|--------|-----------|-----------|-----------|
| 카카오 | Playwright (headless) | `page.evaluate()` JS — React SPA 대응 | 없음 |
| 다이소 | Playwright (headless) | `page.evaluate()` JS — "실시간 랭킹" 텍스트로 섹션 탐색 | 없음 |
| 올리브영 | **curl_cffi** (Chrome TLS 핑거프린트 모방) | regex — `tx_brand`, `tx_name`, `tx_cur`, `t_number` 파싱 | **IP 차단 → TLS 모방으로 우회** |

### 올리브영 우회 원리
- 일반 `requests` → 403 (Cloudflare IP 차단)
- `curl_cffi` + `impersonate="chrome124"` → **200 성공**
- Chrome의 TLS 핑거프린트(JA3/JA4)를 완벽히 모방하여 봇 감지 통과
- GitHub Actions(Azure IP)에서도 동일하게 작동 → **PC 없이 완전 자동화**

## 7. 자동화 (GitHub Actions)

| 워크플로 | 트리거 | 동작 |
|---------|--------|------|
| `daily_crawl.yml` | 매일 UTC 00:00 (= KST 09:00) + 수동 | 3개 플랫폼 수집 → xlsx 저장 → git push |
| `monthly_aggregate.yml` | 매월 1일 UTC 00:00 + 수동 | 전월 daily xlsx 집계 → 월별 xlsx 저장 → git push |

**실행 환경:** `ubuntu-22.04` (ubuntu-latest=24.04는 Playwright libasound2 호환 문제)

**수동 실행 방법:**
1. https://github.com/primexx98-sudo/Health-Ranking/actions 접속
2. 왼쪽 워크플로 선택 → **Run workflow** 클릭

**실패 시 확인:**
- Actions 로그에서 오류 메시지 확인
- `screenshots/` 폴더가 Artifact로 업로드됨 (실행 결과 하단에서 다운로드)

## 9. 엑셀 파일 구조

### 일별 파일 `data/daily/YYYY-MM-DD.xlsx`
- 시트 3개: **카카오선물하기** / **다이소몰** / **올리브영**
- 카카오 시트: 건강식품·영양제(1~10위) + 다이어트·이너뷰티(1~10위) 합계 최대 20행

### 월별 파일 `data/monthly/YYYY-MM_월별취합.xlsx`
- 시트 3개: **카카오선물하기** / **다이소몰** / **올리브영**
- 컬럼: `순위(월평균) | 상품명 | 브랜드 | 평균가격 | 평균순위 | 등장횟수`
- 정렬: 평균순위 오름차순 TOP 10만 표시

### 월별 집계 로직
```
월평균순위 = SUM(해당월 일별순위) / 등장일수
등장횟수   = 해당월에 TOP10 안에 든 날 수
월간TOP10  = 평균순위 기준 상위 10개
```
순위에 없던 날은 제외 (결측일을 11위로 처리하지 않음)

## 11. 유지보수 가이드

| 상황 | 조치 |
|------|------|
| 사이트 URL 변경 | `crawlers/config.py` → 해당 플랫폼 URL 수정 |
| 카카오/다이소 셀렉터 변경 | 각 크롤러 파일 내 `_JS` 변수 수정 |
| 올리브영 HTML 구조 변경 | `crawlers/oliveyoung.py` → `_BRAND_RE`, `_NAME_RE`, `_PRICE_RE` regex 수정 |
| curl_cffi Chrome 버전 구버전 감지 | `impersonate="chrome124"` → 최신 버전으로 변경 |
| GitHub Actions push 실패 | PAT 만료 확인 → 재발급 후 remote URL 업데이트 |
| Actions libasound2 오류 | `runs-on: ubuntu-22.04` 유지 (ubuntu-latest 사용 금지) |
