최초 작성: 2026-06-25  
최종 수정: 2026-06-25 (Streamlit 로컬 앱 추가)

---

## 1. 프로젝트 개요

3개 플랫폼(올리브영, 다이소몰, 카카오 선물하기)의 건강기능식품 TOP10 판매순위를
매일 자동 수집하여 일별 엑셀로 저장하고, 월말에 평균 순위 기준 월간 TOP10을 집계한다.

## 4. 운영 방식

| 경로 | 대상 플랫폼 | 실행 방법 |
|------|------------|-----------|
| GitHub Actions | 카카오 + 다이소 + **올리브영** | 매일 09:00 KST 자동 실행 (PC 꺼진 상태에도 동작) |
| Streamlit 로컬 앱 (`app.py`) | 카카오 + 다이소 + 올리브영 | PC에서 수동 실행 |

> 올리브영은 `curl_cffi`로 Chrome TLS 핑거프린트를 모방하여 Cloudflare를 우회 → GitHub Actions(Azure IP)에서도 수집 가능

## 5. 자동화 방식 (GitHub Actions)

- **플랫폼:** GitHub Actions (클라우드, PC 비활성 상태에도 실행)
- **실행 환경:** ubuntu-22.04 (ubuntu-latest는 24.04로 Playwright libasound2 호환성 문제 있음)
- **실행 주기:** 매일 오전 9시 (KST = UTC 00:00)
- **실패 알림:** GitHub 이메일 알림 (Actions 실패 시 자동 발송)

## 6. 엑셀 파일 구조

### 일별 파일 (`data/daily/YYYY-MM-DD.xlsx`)
- 시트 3개: 카카오선물하기 / 다이소몰 / 올리브영
- 각 시트 컬럼: `카테고리 | 순위 | 상품명 | 브랜드 | 가격 | 상품URL`

### 월별 파일 (`data/monthly/YYYY-MM_월별취합.xlsx`)
- 시트 3개: 카카오선물하기 / 다이소몰 / 올리브영
- 각 시트 컬럼: `순위(월평균) | 상품명 | 브랜드 | 평균가격 | 평균순위 | 등장횟수`
- 정렬 기준: 평균순위 오름차순 → TOP10만 표시

---

## 7. 월별 집계 로직

```
월평균순위 = SUM(해당월 일별순위) / 등장일수
등장횟수 = 해당월에 TOP10 안에 든 날 수
최종 월간 TOP10 = 평균순위 기준 상위 10개 상품
```

> 순위에 없던 날은 제외하고 평균 계산 (결측일을 11위로 처리하지 않음)

## 8. Streamlit 로컬 앱 (`app.py`)

- **실행:** `streamlit run app.py` → `localhost:8501` 브라우저로 열림
- **올리브영:** `headless=False` — Chromium 창이 화면에 열림, 집 IP라 CF 자동 통과 기대
- **GitHub Push 버튼:** 수집 완료 후 결과를 리포에 직접 push
- **설치:** `pip install streamlit playwright-stealth && playwright install chromium`

| 엑셀 저장 | openpyxl |

## 9. 크롤러 방식 (플랫폼별)

| 플랫폼 | 방식 | 비고 |
|--------|------|------|
| 다이소몰 | `page.evaluate()` JS 추출 | "실시간 랭킹" 텍스트로 섹션 탐색 후 li 추출 |
| 카카오 선물하기 | `page.evaluate()` JS 추출 | React SPA, 추천순 첫 10개를 순위로 취급 |
| 올리브영 | `curl_cffi` (Chrome TLS 핑거프린트 모방) + regex | Cloudflare IP 차단 우회, PC 꺼진 상태에서도 GitHub Actions 자동 실행 가능 |

## 10. GitHub Actions 워크플로

### daily_crawl.yml
- **트리거:** 매일 UTC 00:00 (KST 09:00) + 수동 실행
- **주요 스텝:** checkout → Python 3.11 → pip install → playwright install chromium → python main.py → git commit & push
- **실패 시:** screenshots/ 폴더를 Artifact로 업로드 (Actions 실행 결과 페이지 하단에서 다운로드 가능)

### monthly_aggregate.yml
- **트리거:** 매월 1일 UTC 00:00 (KST 09:00) + 수동 실행
- **주요 스텝:** checkout (전체 히스토리) → pip install → python monthly_aggregate.py → git commit & push
